## 🧠 ML Model Info

- **Algorithm Used**: Logistic Regression
- **Dataset Size**: 1,000 records
- **Features**:
  - Gender, Married, Dependents, Education  
  - Self_Employed, Applicant Income, Coapplicant Income  
  - LoanAmount, Loan_Amount_Term, Credit_History  
  - Property_Area
